REST API endpoints


The REST API za pristup podacima o saobraćajnim nezgodama Ministarstva Unutrašnjih Poslova Srbije. Pomoću ovog API-ja moguće je dobiti podatke o svim nezgodama na teritoriji grada Beograda, statistikama, planeru, predviđanjima...

Odgovori su u poznatom JSON formatu, i korišćenje API-ja je potpuno besplatno.
